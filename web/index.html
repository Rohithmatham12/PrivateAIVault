PrivateAIVault

AI redaction + Midnight zero-knowledge commitments. Sensitive data is processed and provably committed, never exposed.

Zero-knowledge · Private by design

Runs entirely in this browser tab. The real compiled Compact circuit executes client-side via WebAssembly, with no server, no Docker, and nothing to keep running.

Step 1 · Raw sensitive record

This text is processed locally in your browser. It is never sent to the Midnight contract or to any server of ours. Only its commitment is.

Optional: paste a Groq API key for real LLM redaction. Left blank, a regex-based redactor runs instead (fully offline, no key needed). The key is used only for a direct browser→Groq call and is never stored or sent anywhere else.

Optional · Real testnet deployment

Everything above runs the real circuit locally, in memory, in this tab. This section instead submits a real, provable transaction to Midnight's public Preprod testnet through a connected wallet. It needs the Lace wallet extension, switched to the Midnight Preprod network, with some free test tokens from its faucet. A transaction can take up to a minute to confirm.
